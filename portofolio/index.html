Pengalaman & Project Magang

Dashboard Rekam Medis

Membangun dashboard rekam medis untuk membantu monitoring operasional dan menyajikan visualisasi data.

Informasi Ketersediaan Bed

Membangun halaman informasi ketersediaan bed secara real-time untuk mendukung layanan operasional pada unit layanan informasi.

Pendaftaran Pasien Online

Membangun halaman sistem pendaftaran pasien online untuk mendukung proses administrasi pendaftaran.

Titip Daftar Pasien Online

Aplikasi Mobile Monitoring Kerja Sama

Aplikasi Mobile Attendance

Membangun aplikasi presensi berbasis Android dengan fitur GPS dan face recoignition untuk mendukung pengelolaan kehadiran.

Kesimpulan

Pengalaman magang sebagai Staf Pengembangan Teknologi Informasi memberikan pengalaman dalam pengembangan dan implementasi sistem informasi untuk mendukung administrasi dan operasional layanan rumah sakit. Kegiatan yang dilakukan meliputi pengembangan sistem berbasis web dan mobile, pengolahan serta penyajian data operasional, pengujian sistem, serta koordinasi dengan pengguna dan tim guna memastikan sistem berjalan sesuai kebutuhan layanan.
